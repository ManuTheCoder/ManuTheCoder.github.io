About me

Hi, I'm Manu, and I'm a junior web developer! I was born in 2008, and I live in the US. I started my journey when I was in fourth grade, doing scratch. I went to a free coding club class in my school, and learnt more about it. When I came to fifth grade, I discovered HTML, CSS, and JavaScript. From then on, I really enjoyed web development! My first project was called "ReadQuest", and online digital learning platform for kids of all ages. Then, my mom inspired me to make what I now present today as Smartlist. In sixth grade, I started to learn PHP basics, and finish up learning JavaScript. In seventh grade, I started learning how to make full-fledged dynamic websites using PHP and SQL. I was also hired to create a website for a business!
